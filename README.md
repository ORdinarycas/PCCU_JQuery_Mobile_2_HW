## 簡報裡的英文關鍵字要記一下
  * Simple Navbar - 簡易導覽列（網站、手機 最上面那條）
  * icon - 圖標
  * Buttons - 按鈕
  * control - 控制
  * group - 群組
  * framework - 骨架、框架
  * List - 清單、列表
  * views - 視圖、檢視、表示、意見
  * content - 內容
  * Formatted - 格式、格式化、制式、標準
  * Split - 切圖、分割、切
  * Thumbnails - 縮略圖、略圖
  * filter - 過濾、篩選
  * dialog - 對話、對話視窗、彈跳視窗
